  --url https://api.vessel.land/crm/users/associations \
  --header 'Content-Type: application/json' \
  --header 'vessel-api-token: '
  --data '{
    "accessToken": "string",
    "objectType": "contact",
    "ids": [
      "v.user.56f653659ea9524b9e0fb0ba67e6f8a0",

  }'

### Response

The response shape depends on the `objectType` parameter.
For example, if `objectType` is `contact`, the response will be

```
{
  "contacts": [
    {
      "id": "v.contact.56f653659ea9524b9e0fb0ba67e6f8a0",
      "nativeId": "23",
      "email": "tom@bluecross.com",
      "firstName": "Tom",
      "lastName": "Smith",
      ...
    },
    ...
  ],
  "nextPageCursor": "string"
}
```
