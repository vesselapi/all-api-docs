The Vessel catalog of integrations. By default all integrations support **at least** authentication handling and passthrough requests. However, some integrations have additional support such as being part of a Unified API or Actions.

## Action Support

The below integrations support 'Actions' and may also be part of a unified API. Actions are a powerful abstraction over the integration's native APIs. You can read more about actions [here](/pages/home/actions).
