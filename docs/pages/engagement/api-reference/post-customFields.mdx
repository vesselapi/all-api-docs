<RequestExample>

```bash Request
curl --request POST \
  --url https://api.vessel.dev/unifications/engagement/contacts/create-custom-field \
  --header 'Content-Type: application/json' \
  --header 'x-vessel-api-token: ' \
  --header 'x-vessel-access-token: '\
  --data '{
  "customField": {
    "name": "customField1",
    "$native": {}
  }
}'
```

</RequestExample>

**Caveats**  
Custom Fields are statically defined in Outreach, use `/engagement/contacts/list-custom-fields` to get a list of pre-defined custom fields.
