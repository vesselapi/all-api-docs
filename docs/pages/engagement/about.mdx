title: "Unified Sales Engagement APIs"
description: "Integrate with Outreach, Salesloft, and more"
---

The unified Sales Engagement APIs allow you to push and pull Accounts, Sequences, Custom Fields, and more from your customer's Sales Engagement Platform.

### Integration Feature Support

| Integration | Unified API | Synced-Cache | Webhooks |
| ----------- | ----------- | ------------ | -------- |
| Apollo      | ✅          | -            | -        |
| Outreach    | ✅          | -            | -        |
| Salesloft   | ✅          | -            | -        |

<Note>
  Don't hesitate to contact `support@vessel.dev` if there are any features you
  need that aren't currently supported. Most features can be added in a matter
  of days.
</Note>
