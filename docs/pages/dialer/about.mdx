---
title: "Unified Dialer APIs"
description: "Integrate with Ringcentral, Aircall, and more"
---

The unified Dialer APIs allow you to push and pull Contacts, Calls, and more from your customer's Dialer.

### Integration Feature Support

| Integration | Unified API | Synced-Cache | Webhooks |
| ----------- | ----------- | ------------ | -------- |
| Aircall     | ✅          | -            | -        |
| Ringcentral | ✅          | -            | -        |
| Dialpad     | ✅          | -            | -        |

<Note>
  Don't hesitate to contact `support@vessel.dev` if there are any features you
  need that aren't currently supported. Most features can be added in a matter
  of days.
</Note>
