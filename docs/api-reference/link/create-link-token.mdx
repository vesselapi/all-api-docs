  })

```python Python
import http.client

conn = http.client.HTTPSConnection("api.vessel.land")

payload = ""

headers = {
    'Content-Type': "application/json",
    'vessel-api-token': ""
    }

conn.request("POST", "/link/token", payload, headers)

res = conn.getresponse()
data = res.read()

print(data.decode("utf-8"))
```
