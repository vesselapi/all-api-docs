```bash Request
  --url https://api.vessel.land/engagement/sequence/step \
  --header 'Content-Type: application/json' \
  --header 'vessel-api-token: ' \
  --data '{
  "accessToken": "string",
  "steps": [{
    "name": "Step 1 (Automated Email)",
    "type": "auto_email",
    "daysAfter": 2,
    "settings": {
      "templateName": "Email Template",
      "isReply": false,
      "subject": "Hello from Dunder!",
      "body": "Hi {{first_name}}! I amm reaching out to ask you about your paper needs."
    },
    "additional": {}
  }]
}'
```

</RequestExample>

**Caveats**

For Salesloft, this endpoint can only be called once, make sure to include all steps in the initial call.
