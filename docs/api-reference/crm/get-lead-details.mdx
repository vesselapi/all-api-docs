---
title: "Get Lead Details"
api: "GET /crm/lead/details"
description: "Get details about all leads."
---

Details include the type, possible values, and other metadata about the fields.

### Parameters

<Param query="accessToken" type="string" required>

The token for the customer's CRM account. This was generated when they connected their account.

</Param>

<Param query="allFields" type="boolean">

Returns details about non-universal fields

</Param>

<Param query="cursor" type="string"></Param>

<Param query="id" type="string">
  If provided, get the details about a particular deal. If not provided, get
  details about all deals.
</Param>
